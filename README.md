
# About AtlasMesh

Dieses Plugin kann die Leistung Ihres Spiels verbessern.

## This package provides:
	1. Unterstützt den Wechsel zwischen verschiedenen Modellen
	2. Unterstützt den Wechsel zwischen verschiedenen Texturen
	3. Unterstützt das Backen von Animationen über das Spine-Plugin
	4. Unterstützt leistungsstarkes Rendering
	5. Unterstützt das Zusammenführen mehrerer Modelle
	6. Unterstützt das Zusammenführen mehrerer Materialie
	7. Unterstützt den Wechsel zwischen verschiedenen Animationen
	8. Unterstützt Animationsschleifen, Geschwindigkeitsanpassung, Einzelwiedergabe usw.

> 1. Supports switching between different models
> 2. Supports switching between different textures
> 3. Supports animation baking via Spine plugin
> 4. Supports high-performance rendering
> 5. Supports merging multiple models
> 6. Supports merging multiple materials
> 7. Supports switching between different animations
> 8. Supports animation looping and speed adjustment, one-shot playback, etc.
